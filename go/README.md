# Go Tutorial

Version: 1.19.4

References:

- Official [documentations](https://go.dev/doc/) and [tutorials](https://go.dev/doc/tutorial/getting-started)

## Install Go on macOS

```bash
brew install go
```

## Examples

### Compile and Run Files

```bash
go build <xxx.go>  # then, ./<xxx>
# Or
go run <xxx.go>
```
